Mini projects using Fusion 360

To start off this new topic we used Fusion 360 to create a very simple object uing very basic methods.

The first mini project we did is a name tag

Name Tag

Steps

  1. Click on create sketch, choose a plane and create a rectangle
  2. Use the fillet tool to round the edges of the sketch
  3. Use the Extrude tool,or press E, and select the sketch
  4. Extrude the rectangle to the desired thickness
  5. Click on create sketch again , click on the top face
  6. drag select everything and go to modify/offset
  7. type in a negative value depending on thick you want the edges to be
  8. Finish sketch , select inner face and extrude downwards
  9. create sketch again and select Text, Choose font , adjust size and type in any text
  10. Extrude text upwards
  11. To make a hole , create new sketch , create a circle and downward extrude
  12. change operation to cut and press ok

Egg Cup

Steps

  1. Create sketch and select centre diam circle , plaace a circle in the origin
  2. choose a plane and make the diameter 38mm
  3. Right click on the circle and click offset plane
  4. move the offset circle by 18mm vertically
  5. Repeat step 1 to 3 but move the offset plane for circle 3 and 4 by 20mm, with diameter 30mm and 44mm respectively
  6. Finish sketch and select the loft tool
  7. Click every circle with new body as its operation
  8. select shell tool and click top circle
  9. Make the thickness 2mm and you are done
